# Ubuntu-ToolKit
This repo is used to install basic essentials for Programming
1. Gcc compiler
2. Python3 , java
3. VS code
4. Chrome
5. wget
6. git
## Installation 
### Method 1
1. Download linux.sh
2. Give permission
   ```bash
   chmod +x linux.sh
3. Run linux.sh
   ```bash
   ./linux.sh

### Method 2(uses GIT)
1. Copy repo in local system
   ```bash
   git clone https://github.com/Akshat-singh90056/Ubuntu-Toolkit
2. move to directory
   ```bash
   cd Ubuntu-Toolkit
3. Run the linux.sh file
   ```bash
   chmod +x linux.sh
   ./linux.sh
